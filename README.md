# The Sampla

A lightweight browser sampler scaffold for hip-hop and trap sample packs.

Live URL after GitHub Pages is enabled:

```txt
https://bearicide.github.io/the-sampla/
```

## Folder map

```txt
assets/
  samples/
    hiphop/
    trap/
  data/
    sample-packs.json
index.html
manifest.webmanifest
sw.js
```

## Current sample-source plan

Actual commercial redistribution of downloaded third-party sample WAV files is not assumed here. The source manifest lists free/royalty-free packs to download manually or import into the folders.

Primary picks:

- 99Sounds Drum Samples — royalty-free multi-genre drum one-shots.
- R-Loops Free Hip-Hop Drum Kits — free WAV hip-hop/trap one-shots and loops.
- FatLoops Alpha — free royalty-free trap/hip-hop sounds and loops.
- FatLoops Boom Bap Drum Kit — free royalty-free boom bap drums.
- SampleRadar Processed 808/909 — royalty-free 808/909 drum material, but do not redistribute the raw ZIP.

## Notes

- Use lowercase, hyphenated filenames.
- Prefer short filenames: `trap-808-a.wav`, `boom-bap-kick-01.wav`, etc.
- Keep source/license notes in `assets/data/sample-packs.json`.
